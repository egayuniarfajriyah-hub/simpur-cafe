# Simpur Cafe Management System

A web-based cafe management system developed using PHP and MySQL.

## Features

* User Authentication (Login & Register)
* Menu Management
* Table Reservation System
* Tenant Management
* Customer Dashboard
* Admin Dashboard
* Reports Management

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript

## User Roles

### Admin

* Manage menu items
* Manage tenants
* Manage reservations
* Generate reports

### Customer

* View menu
* Make reservations
* View reservation history

### Tenant

* Manage menu items
* Update menu information

## Database

MySQL database included in:
`database/simpur_cafe.sql`
